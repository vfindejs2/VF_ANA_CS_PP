Systémové služby 

Databázový systém 

Primárny databázový systém MS SQL Server vo verzii 17. Dátový model je popísaný tu https://confluence.radium.cz/pages/viewpage.action?pageId=88033012

REST API (backend) 

Poskytuje REST API pre FE aplikáciu a zabezpečuje komunikáciu s externými službami FLWW

Backend je postavený na technológii NodeJS, verzia 12

Sync Service 

Všeobecná synchronizačná služba na ktorú sa integrujú synchronizačné služby pre konkrétny externý systém. Popis tu Synchronizácia dát z externých systémov

Postavený na technológii NodeJS, verzia 12

Winyx Sync Service 

Služba zabezpečujúca synchronizáciu dát z externého systému WinyX. Detailný popis tu WinyX Sync Service
