Nastavení čísla sledovaného vstupu

Aby RoadPlan správně vyhodnocoval aktivace vstupu vozidla, je zapotřebí příslušnému vozidlu určit daný vstup pomocí labelu, jehož klíč odpovídá konfigurační hodnotě PTOInputLabelKey.

 Pokud není label explicitně nastavený, je používán vstup definovaný konfiguračním parametrem defaultPTOInput. (V MPG SK se používá vstup číslo 6.)

V současné době nelze label nastavit z UI.

Nastavení z API
POST FleetwareWebAPI/objects//labels/
{
"key": "PTOInput",
"value": ""
}

Příklad volání API (PROD MPG SK)
POST https://mpg.fleetwareweb.eu/FleetwareWebAPI/objects/9797/labels/PTOInput
{
"key": "PTOInput",
"value": "5"
}

Nastavení přímo na databázi
insert into [Fleetware].[dbo].[ObjectLabel]
([ObjectId],[ObjectLabelKey],[ObjectLabelValue])
values (,'PTOInput','')
