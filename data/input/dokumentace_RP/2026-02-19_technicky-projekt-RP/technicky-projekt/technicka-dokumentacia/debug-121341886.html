Debug nastroje su interne tools systemu na riesenie roznych incidentov, spustanie specifickych sluzieb, jobov a pod.

Viacere nastroje sa ovladaju cez REST API pre backend komponent. V detaile konkretneho nastroja je URL na backend uvedene ako {{url-backend}}. Napr. pre prostredie rptestradium2, je URL: https://mpg-rptestradium2.fleetwareweb.eu/apiBackend

  • Takisto je pouzita basic autentifikacia. Pri pouziti API treba meno a heslo vyziadat od ai-maps.
