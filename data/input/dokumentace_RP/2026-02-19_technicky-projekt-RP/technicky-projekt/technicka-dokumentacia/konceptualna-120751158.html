Diagram

Diagram znazornuje prehlad vsetkych komponentov Roadplan. Pre konkretne nasadenie RP pre zakaznika sa ale nenasadzuju vsetky komponenty, tam sa vyberaju podla integracie s externym systemom (vid Zákaznícke špecifiká)

 

 

Prehlad komponentov

Komponent

Popis

Poznamka

frontend

Web aplikacia frontendovej casti systemu

 

backend

poskytuje REST API pre Frontend a predstavuje jadro systemu

 

database

MS SQL Server na ukladanie udajov

 

sync service

Vseobecna synchronizacna sluzba na ukladanie udajov z externych systemov, pricom s touto sluzbou komunikuju specificke synchronizacne sluzby

 

OSRM

Open Street Routing Machine - open source sluzba pre routing

 

browserless

komponent sluziaciaci na vygenerovanie PDF z HTML obsahu

 

mpg-flww-sync-service

sync sluzba na ukladanie udajov z FLWW. Je specificka pre zakaznika MPG

 

recovera-flww-sync-service

sync sluzba na ukladanie udajov z FLWW. Je specificka pre zakaznika Recovera

 

mpg-sync-service

specificka sync sluzba na ziskanie udajov z Winyx DB pre zakaznika MPG

 

mpg-helios-sync-service

specificka sync sluzba na ziskanie udajov z Helios DB pre zakaznika MPG

 

recovera-sync-service

specificka sync sluzba na ziskanie udajov z Winyx API pre zakaznika Recovera
